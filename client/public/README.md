# Favicon para PatriPsicología

Este directorio contiene los archivos de favicon para la aplicación PatriPsicología. Para actualizar con tu logo personalizado:

## Cómo reemplazar el favicon

1. Prepara tu logo en los siguientes formatos y tamaños:
   - `favicon.ico` - Formato ICO (recomendado 32x32)
   - `favicon-32x32.png` - PNG de 32x32 píxeles
   - `favicon-16x16.png` - PNG de 16x16 píxeles
   - `apple-touch-icon.png` - PNG de 180x180 píxeles (para dispositivos Apple)

2. Reemplaza los archivos existentes con tus imágenes personalizadas.

## Generación de favicon

Puedes usar herramientas online como:
- [Favicon.io](https://favicon.io/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)

Estas herramientas te permiten subir tu logo y generar automáticamente todos los formatos necesarios.

## Notas

- Asegúrate de que tu logo sea claramente visible incluso en tamaños pequeños
- El formato ICO es especialmente importante para compatibilidad con navegadores más antiguos
- El archivo `favicon.png` sigue siendo necesario para el soporte directo en algunos navegadores
